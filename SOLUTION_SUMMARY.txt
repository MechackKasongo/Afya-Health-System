PROBLEM:
--------

  "Detected failed migration to version 11 (refresh tokens user fk)"
Root Cause: V11 migration failed when encountering orphaned refresh_tokens 
rows with no matching app_users. It raised a hard error instead of handling 
gracefully, causing Flyway to mark the migration as FAILED and block startup.
